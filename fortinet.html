Fortinet Security Fabric

FortiGate firewalls, FortiSwitch, FortiAP access points, FortiWLC, FortiManager, and the integrated Security Fabric ecosystem.

ℹ️

FortiOS: All FortiGate devices run FortiOS. CLI uses a hierarchical config tree: config system interface / edit port1 / set ip 192.168.1.1/24 / end. Key concept: FortiLink — a special trunk port that connects a FortiGate to FortiSwitches/FortiAPs, allowing the FortiGate to manage them. Certifications: NSE 1 (free, online) → NSE 4 (associate) → NSE 7/8 (expert).

FortiManager & FortiAnalyzer

🖥️

FortiManager

Centralized policy management for multiple FortiGate firewalls, FortiSwitches, and FortiAPs. Create policy packages and push to hundreds of devices. Available as hardware appliance or VM.

  • ADOMs — Administrative Domains for multi-tenant
  • Policy packages pushed to device groups
  • Zero-touch provisioning for new devices
  • Manages FortiGate, FortiSwitch, FortiAP, FortiWLC
📊

FortiAnalyzer

Centralized log aggregation, analytics, and reporting for all Fortinet products. Correlates events across firewalls, switches, APs, and endpoints to detect threats and demonstrate compliance.

  • Receives logs from all FortiGate devices
  • Security incidents, compliance reports
  • SIEM-lite: correlation rules + alerts
  • Available as hardware or VM (Azure/AWS)
